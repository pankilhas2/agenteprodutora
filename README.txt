# Agente IA - Áudio Pank (Corrigido)

## Instruções para usar na Vercel:

1. Faça o upload do projeto para o GitHub.
2. Na Vercel:
   - Configure o repositório.
   - Em "Root Directory", coloque: . (ponto)
   - Em "Build Command": npm run build
   - Em "Install Command": npm install
   - Em "Output Directory": .next

## Rodar localmente:
npm install
npm run dev
